Detailed Description

Datos base de tetrominós siguiendo SRS para la lógica del juego. Define formas iniciales, rotación de 90° y tablas de wall kicks.

Convenciones

  • Coordenadas en celdas (Vector2Int), relativas al origen/pivote de la pieza.
  • Sistema de ejes de Unity: +X a la derecha, +Y hacia arriba.
  • Rotaciones de 90° en sentido antihorario (CCW), salvo indicación contraria.

Estructuras

  • RotationMatrix: matriz 2D para girar 90° CCW.
  • Cells: forma inicial (spawn) de cada tetrominó, 4 celdas relativas.
  • WallKicks: tablas SRS de corrección (dx, dy) por tipo de pieza y transición de rotación.

Matriz de rotación 2D (90° CCW) en orden fila-mayor: [ cos sin ; -sin cos ].

El arreglo se almacena como { m00, m01, m10, m11 }. Aplicación a un vector (x, y): x' = m00*x + m01*y, y' = m10*x + m11*y.
